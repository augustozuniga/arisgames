createItemAndGiveToPlayer

public static returnData createItemAndGiveToPlayer(integer gameId, integer playerId, string name, string description, string fileName, bool droppable, bool destroyable, float latitude, float longitude)

Create an item, simultaneously creating media from a file, linking it to the new media and giving it to a player.

This service combines several calls into one, avoiding async delays caused by data dependancies created while waiting for createItem() and createMedia() to return the new item/media id

Parameters:
gameId - The game identifier
playerId - The player identifier
name - The name
description - The html formatted description
fileName - The name of the file in the game's data directory that will act as this item's media
droppable - 1 if this item can be dropped, 0 if not
destroyable - 1 if this item can be detroyed, 0 if not
latitude - Latitude where item was created
longitude - Longitude where item was created
Returns:
a returnData object containing the new item identifier
